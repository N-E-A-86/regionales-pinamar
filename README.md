# 📸 Regionales Pinamar - Galería de Recuerdos

**Regionales Pinamar** es una plataforma web interactiva diseñada para capturar y compartir la experiencia de los visitantes en el icónico monumento al Mate Gigante en Pinamar, Argentina. El proyecto fomenta la creación de una comunidad digital, permitiendo a los turistas inmortalizar su paso por la tienda y participar en beneficios exclusivos.

---

## 🚀 Características del Proyecto

### MVP (Producto Mínimo Viable) - Versión Actual
* **Gestión de Usuarios:** Registro y autenticación segura para la interacción comunitaria.
* **Feed Fotográfico:** Galería dinámica con sistema de moderación para asegurar contenido de calidad.
* **Interacción Social:** Sistema integrado de "Me gusta" y comentarios en tiempo real.
* **Página Individual de Captura:** Visualización detallada de fotos con metadatos (usuario, fecha, interacción).
* **Sección de Promociones:** Panel dinámico para la fidelización de clientes mediante beneficios en tienda física.
* **Administración Centralizada:** Panel de control para moderación de contenido y gestión de promociones.

### Roadmap (Próximas Implementaciones)
* **Social Auth:** Login integrado con Google y Facebook.
* **Social Sharing:** Botones directos para compartir en Instagram y WhatsApp.
* **Gamificación:** Concursos semanales por "La Foto del Mes" y notificaciones push de logros.
* **User Profiles:** Perfiles públicos con la colección personal de cada turista.

## 🛠️ Stack Tecnológico

El proyecto está construido bajo una arquitectura robusta y escalable:

* **Backend:** [Laravel](https://laravel.com/) (PHP 8.x) - Potenciando la lógica de negocio y la seguridad.
* **Frontend:** Blade Templates, CSS3, JavaScript (modernización progresiva).
* **Base de Datos:** MySQL - Gestión de datos relacionales y consistencia de información.
* **Almacenamiento Cloud:** [Google Cloud Platform (GCP)](https://cloud.google.com/) - Hosting eficiente de activos multimedia.
* **Infraestructura:** [Hostinger](https://www.hostinger.com/) - Servidor optimizado para despliegue en producción.
